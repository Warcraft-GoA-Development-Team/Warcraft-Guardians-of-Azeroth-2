# Note: Longevity Flags by SS
# Since I can't figure out how to access the life expectancy value, I'm instead going to add flags to these traits based on how much LE bonus they provide
# Each tier of life expectancy flag represents one order of magnitude for bonus LE [(10^f)*c = e; where f is the flag value, c is a flavor value (3.3 for gnomes, 1.5 for high elves, etc), and e = life expectancy]
# life_expectancy_0 = no added value
# life_expectancy_1 = anything up to 100 [Tauren, Taunka, etc]
# life_expectancy_2 = anything 100 to 1000 [gnomes, dwarves, vrykul, etc]
# life_expectancy_3 = anything 1000 to 10000 [elves etc]
# life_expectancy_4 = anything 10000 + [dragons, titan constructs, etc]
# For more information on how this system works, please check out the feature documentation on google drive, titled "Feature: Longevity Mechanics"
